# Template: Agent Description

Use this template to generate the display name and descriptions for a Copilot Studio agent.

---

## Fields to Generate

### Display Name
- Max 30 characters
- Format: [Domain] + "Assistant" or "Agent"
- Must be immediately clear what the agent does

### Short Description
- Max 250 characters
- One sentence that explains what the agent does and for whom
- Appears in Copilot Studio agent catalog and Teams app store

### Long Description
- 2-4 sentences
- What the agent does, what topics it covers, and what it escalates
- Used for internal documentation and onboarding

---

## Examples

| Field | IT Agent | HR Agent |
|---|---|---|
| Display Name | IT Support Assistant | HR Policy Assistant |
| Short Description | Helps employees troubleshoot IT issues with step-by-step guidance. Covers password resets, software installs, VPN, and device setup. Escalates security and hardware issues to IT support. | Helps employees find answers to HR questions about policies, benefits, leave, and onboarding. Escalates formal complaints, legal matters, and accommodation requests to the People team. |
| Long Description | The IT Support Assistant guides employees through common technical issues including password resets, software installation, VPN configuration, and device setup. It provides numbered step-by-step instructions and links to relevant KB articles. For security incidents, hardware failures, or admin privilege requests, it escalates to the IT support team. | The HR Policy Assistant helps employees navigate company policies, benefits enrollment, leave requests, and onboarding procedures. It provides authoritative guidance based on the employee handbook and HR knowledge base, personalized by country and role. For formal complaints, accommodation requests, or situations involving legal risk, it connects employees with the People team. |
