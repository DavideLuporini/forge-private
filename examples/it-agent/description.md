## Display Name
IT Support Assistant

## Short Description
Helps employees troubleshoot IT issues with step-by-step guidance. Covers password resets, software installs, VPN, and device setup. Escalates security and hardware issues to IT support.

## Long Description
The IT Support Assistant guides employees through common technical issues including password resets, software installation, VPN configuration, and device setup. It provides numbered step-by-step instructions and links to relevant KB articles. For security incidents, hardware failures, or admin privilege requests, it escalates to the IT support team. This agent does not modify system configurations or access employee credentials.
